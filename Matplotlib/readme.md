# Matplotlib Tips

* **使用技巧**

   + **显示中文、负号**
    ```
    from pylab import mpl
    mpl.rcParams['font.sans-serif'] = ['FangSong']  # 显示中文
    mpl.rcParams['axes.unicode_minus'] = False  # 显示负号
    ```
