# 基于sqlite3的SQL语句训练

建立本地数据库，实现SQL语句的学习。

 1. 安装Python包**sqlite3**，可利用**Anconda**安装。
```
    import sqlite3
    # 打印版本
    print('版本号：%s' % sqlite3.sqlite_version)
```
执行以上语句，不报错，说明安装成功。
 

 2. 安装**Navicat for SQLite**，安装完成后，点击Connection——New SQLite 3，在Database File选择本地的数据库地址。
